# 📖 Project Overview

LinkUps is a modern **AI-powered professional networking platform** designed to bridge the gap between traditional social networking and career development.

Unlike conventional networking applications, LinkUps integrates **Google Gemini AI**, **real-time communication**, **resume engineering**, and **AI-assisted interview preparation** into one unified platform.

Whether users want to build professional connections, optimize resumes, prepare for interviews, or collaborate through secure video meetings, LinkUps provides an intelligent ecosystem that enhances every stage of their career journey.

The platform follows a **production-ready MERN architecture** with **Next.js**, **Express.js**, **MongoDB**, **Socket.IO**, **WebRTC**, and **Google Gemini**, ensuring scalability, responsiveness, and maintainability.

# 🧠 AI Career Suite

At the heart of **LinkUps** is an intelligent career assistant powered by **Google Gemini**, designed to help users prepare for interviews, optimize resumes, and improve job application success rates.

### 🚀 AI Resume-to-JD Tailoring

Automatically analyzes a target Job Description (JD) and intelligently rewrites the resume to maximize relevance.

**Capabilities**

- ATS-friendly resume optimization
- Keyword matching
- Experience prioritization
- Project enhancement
- Professional summary rewriting
- Skill gap identification

---

### 📊 ATS Match Score Analyzer

Evaluate how well a resume aligns with a target job description.

The AI provides:

- ATS Compatibility Score (0–100)
- Matching Skills
- Missing Keywords
- Resume Weaknesses
- Improvement Suggestions
- Hiring Probability Insights

---

### 🎤 AI Mock Interview

Generate personalized interview sessions tailored to:

- Resume

# ⚙️ Local Installation & Setup

## 📋 Prerequisites

Before running the project locally, ensure you have the following installed:

- Node.js (v18 or later)
- npm or Yarn
- MongoDB Atlas (or Local MongoDB)
- Git

- Google Gemini API

- Google OAuth
- GitHub OAuth

- ElevenLabs

# 📥 Clone Repository

```bash
git clone https://github.com/Utkal9/linkups.git

cd linkups
```

---

# 📦 Backend Setup

Navigate to the backend directory

```bash
cd backend
```

Install dependencies

```bash
npm install
```

Create

```
.env
```

```env
PORT=9090

MONGO_URL=

SESSION_SECRET=

BACKEND_URL=http://localhost:9090

FRONTEND_URL=http://localhost:3000

GOOGLE_CLIENT_ID=
GOOGLE_CLIENT_SECRET=

GITHUB_CLIENT_ID=
GITHUB_CLIENT_SECRET=

GEMINI_API_KEY=

DID_API_KEY=
DID_PRESENTER_URL=

ELEVENLABS_API_KEY=
ELEVENLABS_VOICE_ID=

CLOUDINARY_CLOUD_NAME=
CLOUDINARY_API_KEY=
CLOUDINARY_API_SECRET=
```

Run

```bash
npm run dev
```

Backend

```
http://localhost:9090
```
